# Site errors

A dedicated stream of everything that failed for this site — deploys, SSL, connectivity, cron, and more — newest first and grouped by cause.

## Triage

Repeated failures collapse into a single grouped entry so the stream stays readable. **Dismiss** what you've handled; **Retry** where the underlying action supports it, such as a failed deploy or certificate issuance.

## Relationship to the server stream

This is the site-scoped slice of the server's **Errors** view — the server stream shows failures across every site it hosts.

## Related sections

- **Deploy** — re-run or inspect a failed deployment
- **Logs** — application and webserver log output
- **Certificates** — resolve SSL issuance failures
