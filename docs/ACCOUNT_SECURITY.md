# Security

**Profile → Security** is where you harden how you sign in: password, passkeys, and two-factor authentication.

## Posture

The hero shows an at-a-glance **posture** for your account:

- **Hardened** — 2FA plus a passkey or OAuth.
- **Good** — password and 2FA.
- **Password only** — add 2FA or a passkey to strengthen it.

## Password

Update your account password from the **Password** card. Use a long, random password stored in a password manager. You confirm your current password and enter the new one twice; unsaved edits are flagged until you **Save password**.

## Passkeys

**Passkeys** let you sign in with your device PIN, fingerprint, or a hardware security key — no password prompt. Multiple passkeys per account are supported, so you can register one per device. Give each a recognisable name (e.g. "Work laptop") when you add it, and remove ones you no longer use from the registered list.

## Two-factor authentication

Enable **2FA** with an authenticator app and enter the generated code to confirm. Once enabled, the hero **2FA** tile reads *Enabled*; otherwise it reads *Off*.
