BaseTask

Foundation class for all tasks in TaskRunner. Provides core functionality for task execution, output handling, and callback support.

Class Signature

abstract class BaseTask

Abstract Methods

handle()

abstract public function handle(): void

Main method that contains the task execution logic. Must be implemented by all task classes.

Configuration Methods

public function setName(string $name): self

Set the task name for identification and logging.

Parameter Type Required Description
$name string Yes Human-readable name for the task

setTimeout()

public function setTimeout(int $seconds): self

Set the maximum execution time for the task in seconds.

Parameter Type Required Description
$seconds int Yes Timeout in seconds

setCallbackUrl()

public function setCallbackUrl(string $url): self

Set the URL for sending HTTP callbacks when task events occur.

Parameter Type Required Description
$url string Yes HTTP URL for callbacks

setCallbackConfig()

Parameter Type Required Description
$config array Yes Configuration array with timeout, max_attempts, delay, etc.

TaskStatus Enum

Defines all possible states a task can be in during its lifecycle.

Enum Values

Active Statuses

  • TaskStatus::Pending - ⏳ Task is waiting to be executed
  • TaskStatus::Running - 🔄 Task is currently executing

Completed Statuses

  • TaskStatus::Finished - ✅ Task completed successfully
  • TaskStatus::Failed - ❌ Task failed during execution
  • TaskStatus::Timeout - ⏰ Task timed out
  • TaskStatus::Cancelled - 🚫 Task was cancelled
  • TaskStatus::UploadFailed - 📤 Task file upload failed
  • TaskStatus::ConnectionFailed - 🔌 Task connection failed

CallbackType Enum

Defines all types of callbacks that can be sent during task execution.

Enum Values

Lifecycle Callbacks

  • CallbackType::Started - Task begins execution
  • CallbackType::Paused - Task is paused
  • CallbackType::Resumed - Task is resumed
  • CallbackType::Cancelled - Task is cancelled

Status Callbacks

  • CallbackType::Progress - Progress update
  • CallbackType::Finished - Successful completion
  • CallbackType::Failed - Execution failure
  • CallbackType::Timeout - Time limit exceeded
  • CallbackType::Custom - Custom callback
